# Contributing to Hollon-AI

Thank you for your interest in contributing! Please follow these simple guidelines:

1. Fork the repository and create a new branch
2. Run `pnpm install` to install dependencies
3. Make your changes following the existing code style
4. Run `pnpm typecheck`, `pnpm lint`, and `pnpm test` before submitting
5. Submit a pull request with a clear description of your changes

For questions or discussions, please open an issue first.
