# Hollon-AI

Hollon-AI is a recursive multi-agent system where AI agents (called "Hollons") collaborate like a startup organization. Each Hollon can autonomously execute tasks, delegate to sub-agents, and participate in team-based workflows with quality gates and escalation policies.

**Tech Stack:** TypeScript, NestJS, PostgreSQL, React

## Getting Started

1. Install dependencies: `pnpm install`
2. Start services: `pnpm docker:up`
3. Run migrations: `pnpm db:migrate`
4. Start server: `pnpm dev`
5. Access API at `http://localhost:3000`
